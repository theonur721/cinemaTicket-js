# Sinema Koltuğu Seçici

Basit bir **JavaScript projesi**. Kullanıcı koltuk seçip film fiyatına göre toplam ücreti görebilir. Seçimler **localStorage**’da saklanır.

## Özellikler

- Koltuk seçme ve çıkarma
- Rezerve koltuklara tıklanamaması
- Film seçimine göre dinamik fiyat
- Seçilen koltuk sayısı ve toplam ücret gösterimi
- localStorage ile seçimlerin kaydedilmesi

## Kütüphaneler / Teknolojiler

- HTML5, CSS3
- Font Awesome (ikon)
- localStorage
